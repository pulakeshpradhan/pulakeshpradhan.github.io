GFS: Global Forecast System

weather forecast
Resolution: 27830m (0.25°)
Period: 2015-07-01 to present
Cadence: 6 Hours
Dataset ID: NOAA/GFS0P25

Available Variables

  • temperature_2m_above_ground Temperature 2m above ground (°C)
  • specific_humidity_2m_above_ground Specific humidity 2m above ground
  • relative_humidity_2m_above_ground Relative humidity 2m above ground (%)
  • dew_point_temperature_2m_above_ground Dew point temperature 2m above ground (°C)
Official Documentation

Code Example

var dataset = ee.ImageCollection('NOAA/GFS0P25')
    .filter(ee.Filter.date('2023-01-01', '2023-01-02'));
var temperature = dataset.select('temperature_2m_above_ground');
var visParams = {
  min: -40.0,
  max: 35.0,
  palette: ['blue', 'purple', 'cyan', 'green', 'yellow', 'red']
};
Map.setCenter(0, 0, 2);
Map.addLayer(temperature, visParams, 'Temperature');
