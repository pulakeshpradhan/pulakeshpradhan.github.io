Introduction to Google Earth Engine

What is Earth Engine?

Google Earth Engine is a cloud-based platform for planetary-scale geospatial analysis that brings Google's computational capabilities to bear on a variety of high-impact societal issues including deforestation, drought, disaster, disease, food security, water management, climate monitoring and environmental protection.

Earth Engine combines a multi-petabyte catalog of satellite imagery and geospatial datasets with planetary-scale analysis capabilities. Scientists, researchers, and developers use Earth Engine to detect changes, map trends, and quantify differences on the Earth's surface.

Official Site

Key Capabilities

  • Access to a multi-petabyte catalog of satellite imagery and geospatial datasets
  • Cloud-based computation for analyzing large datasets
  • APIs for JavaScript and Python
  • Interactive development environment
  • Integration with other Google services

Earth Engine is designed to empower not only traditional remote sensing scientists, but also a much wider audience that lacks the technical capacity needed to utilize traditional supercomputers or large-scale commodity cloud computing resources.

Data Catalog

Earth Engine hosts satellite imagery and stores it in a public data archive that includes historical earth images going back more than forty years. The images are ingested on a daily basis and made available for global-scale data mining.

The archive includes data from various satellite platforms like Landsat, Sentinel, MODIS, NAIP, and more, as well as geophysical datasets for climate, weather, and terrain.

Browse Datasets

Available Datasets

  • Landsat: Collection of optical imagery from Landsat missions (1-9)
  • Sentinel: Optical and radar imagery from the European Space Agency
  • MODIS: Moderate Resolution Imaging Spectroradiometer data
  • Climate: Temperature, precipitation, and climate model data
  • Terrain: Digital elevation models and terrain derivatives
  • Land Cover: Global and regional land cover maps

Earth Engine's data catalog is constantly growing, with new datasets being added regularly. Users can also upload their own raster or vector data to use in their analyses.

Cloud Computing

Powerful

Earth Engine provides a cloud-based computational platform that allows users to run algorithms on Google's infrastructure. This enables the analysis of very large datasets without having to download them or worry about computational capacity.

The platform uses a distributed computing approach, where computations are split across many computers in Google's data centers, allowing for efficient processing of petabyte-scale datasets.

Scale Guide

Computational Advantages

  • Parallel processing across thousands of computers
  • Lazy computation model for efficiency
  • Server-side processing eliminates need to download data
  • Automatic scaling based on computational needs
  • Optimized algorithms for geospatial analysis

Earth Engine's computational model is designed to handle the unique challenges of geospatial data processing, including dealing with projections, spatial operations, and large raster datasets.

Key Features & Capabilities

Image Processing

Perform operations on satellite imagery including filtering, masking, compositing, and spectral transformations. Apply algorithms to entire image collections efficiently.

Time Series Analysis

Analyze changes over time using multi-temporal datasets. Track vegetation changes, urban growth, deforestation, and other dynamic processes.

Machine Learning

Use built-in machine learning algorithms for classification, regression, and clustering. Train models on satellite imagery for land cover mapping and change detection.

Vector Analysis

Work with vector data for area calculations, buffering, and zonal statistics. Integrate with raster data for comprehensive spatial analysis.

Visualization

Create interactive maps and visualizations of analysis results. Export images, charts, and data for further use or publication.

API Access

Access Earth Engine through JavaScript or Python APIs. Integrate with other tools and workflows for comprehensive geospatial analysis.

JavaScript API

Getting Started

The Earth Engine JavaScript API is the primary way to interact with Earth Engine. It provides a comprehensive set of functions for accessing and analyzing geospatial data in the Earth Engine catalog.

// Load an image var image = ee.Image('LANDSAT/LC08/C02/T1_TOA/LC08_044034_20140318'); // Print image metadata to the console print(image); // Calculate NDVI var ndvi = image.normalizedDifference(['B5', 'B4']).rename('NDVI'); // Define visualization parameters var visParams = { min: 0, max: 1, palette: ['red', 'yellow', 'green'] }; // Add the NDVI layer to the map Map.centerObject(image, 9); Map.addLayer(ndvi, visParams, 'NDVI'); // Add the natural color image to the map Map.addLayer(image, {bands: ['B4', 'B3', 'B2'], min: 0, max: 0.3}, 'Landsat 8'); // Export the NDVI image to your Google Drive Export.image.toDrive({ image: ndvi, description: 'NDVI_Image', scale: 30, region: image.geometry(), maxPixels: 1e9 });

Code Editor

Recommended

The Earth Engine Code Editor is a web-based integrated development environment (IDE) for the Earth Engine JavaScript API. It provides a convenient interface for writing, running, and visualizing Earth Engine analyses.

// Define a region of interest var roi = ee.Geometry.Point([-122.0841, 37.4223]).buffer(10000); // Load Landsat 8 collection var collection = ee.ImageCollection('LANDSAT/LC08/C02/T1_TOA') .filterDate('2020-01-01', '2020-12-31') .filterBounds(roi); // Create a cloud-free composite var composite = collection.median(); // Add layers to the map Map.centerObject(roi, 10); Map.addLayer(composite, {bands: ['B4', 'B3', 'B2'], min: 0, max: 0.3}, 'RGB'); Map.addLayer(composite, {bands: ['B5', 'B4', 'B3'], min: 0, max: 0.3}, 'False Color'); // Add drawing tools to the map Map.drawingTools().setShown(true); // Create a panel with widgets var panel = ui.Panel({ style: { width: '300px', position: 'top-right' } }); // Add a title to the panel panel.add(ui.Label({ value: 'Landsat 8 Explorer', style: {fontSize: '24px', fontWeight: 'bold'} })); // Add a dropdown to select visualization var select = ui.Select({ items: ['Natural Color (4,3,2)', 'False Color (5,4,3)', 'NDVI'], placeholder: 'Select visualization', onChange: function(value) { if (value === 'Natural Color (4,3,2)') { Map.layers().set(0, ui.Map.Layer(composite, {bands: ['B4', 'B3', 'B2'], min: 0, max: 0.3}, 'RGB')); } else if (value === 'False Color (5,4,3)') { Map.layers().set(0, ui.Map.Layer(composite, {bands: ['B5', 'B4', 'B3'], min: 0, max: 0.3}, 'False Color')); } else if (value === 'NDVI') { var ndvi = composite.normalizedDifference(['B5', 'B4']).rename('NDVI'); Map.layers().set(0, ui.Map.Layer(ndvi, {min: 0, max: 1, palette: ['red', 'yellow', 'green']}, 'NDVI')); } } }); panel.add(select); // Add the panel to the map Map.add(panel);

UI Components

Earth Engine provides a set of user interface (UI) components that can be used to create interactive applications in the Code Editor. These components include buttons, sliders, dropdowns, and panels that can be combined to create custom interfaces.

// Create a panel to hold widgets var panel = ui.Panel({ style: { width: '300px', padding: '10px' } }); // Add a title panel.add(ui.Label({ value: 'NDVI Explorer', style: {fontSize: '20px', fontWeight: 'bold', margin: '10px 0'} })); // Add a date slider var startDate = '2020-01-01'; var endDate = '2020-12-31'; var dateSlider = ui.DateSlider({ start: startDate, end: endDate, period: 30, // days style: {width: '280px'}, onChange: function(range) { updateMap(range.start(), range.end()); } }); panel.add(ui.Label('Select date range:')); panel.add(dateSlider); // Add a region selector var regions = { 'San Francisco': [-122.4194, 37.7749], 'New York': [-73.9352, 40.7306], 'Tokyo': [139.6917, 35.6895], 'London': [-0.1278, 51.5074] }; var regionSelect = ui.Select({ items: Object.keys(regions), placeholder: 'Select a region', onChange: function(key) { var coords = regions[key]; Map.setCenter(coords[0], coords[1], 10); updateMap(dateSlider.getValue().start(), dateSlider.getValue().end()); } }); panel.add(ui.Label('Select region:')); panel.add(regionSelect); // Add a checkbox for cloud masking var cloudCheckbox = ui.Checkbox({ label: 'Apply cloud masking', value: true, onChange: function(checked) { updateMap(dateSlider.getValue().start(), dateSlider.getValue().end()); } }); panel.add(cloudCheckbox); // Function to update the map function updateMap(start, end) { var region = regions[regionSelect.getValue()]; if (!region) return; var point = ee.Geometry.Point(region); var collection = ee.ImageCollection('LANDSAT/LC08/C02/T1_TOA') .filterDate(start, end) .filterBounds(point); if (cloudCheckbox.getValue()) { collection = collection.filter(ee.Filter.lt('CLOUD_COVER', 20)); } var composite = collection.median(); var ndvi = composite.normalizedDifference(['B5', 'B4']).rename('NDVI'); Map.layers().reset(); Map.addLayer(composite, {bands: ['B4', 'B3', 'B2'], min: 0, max: 0.3}, 'RGB'); Map.addLayer(ndvi, {min: 0, max: 1, palette: ['red', 'yellow', 'green']}, 'NDVI'); } // Add the panel to the map ui.root.add(panel);

Code Examples

Landsat Analysis

Landsat satellites have been collecting imagery of Earth since 1972, providing one of the longest continuous records of Earth's land surface. Earth Engine provides access to the entire Landsat archive.

// Define a region of interest var roi = ee.Geometry.Point([-122.0841, 37.4223]).buffer(50000); // Get Landsat 8 collection var collection = ee.ImageCollection('LANDSAT/LC08/C02/T1_TOA') // Filter by date and region .filterDate('2020-01-01', '2020-12-31') .filterBounds(roi); // Create a cloud-free composite // This uses the median value of all available images var composite = collection.median(); // Define visualization parameters var visParams = { bands: ['B4', 'B3', 'B2'], // Red, Green, Blue min: 0, max: 0.3 }; // Center the map on the region of interest Map.centerObject(roi, 8); // Add the composite to the map Map.addLayer(composite, visParams, 'Landsat 8 Composite'); // Add a false color composite (NIR, Red, Green) var visFalseColor = { bands: ['B5', 'B4', 'B3'], min: 0, max: 0.3 }; Map.addLayer(composite, visFalseColor, 'False Color Composite');
// Define a region of interest (point or polygon) var point = ee.Geometry.Point([-122.0841, 37.4223]); // Get Landsat 8 collection var collection = ee.ImageCollection('LANDSAT/LC08/C02/T1_TOA') // Filter by date and region .filterDate('2018-01-01', '2022-12-31') .filterBounds(point); // Function to calculate NDVI function addNDVI(image) { var ndvi = image.normalizedDifference(['B5', 'B4']).rename('NDVI'); return image.addBands(ndvi); } // Add NDVI band to each image var withNDVI = collection.map(addNDVI); // Create a time series chart var chart = ui.Chart.image.series({ imageCollection: withNDVI.select('NDVI'), region: point, reducer: ee.Reducer.mean(), scale: 30 }).setOptions({ title: 'NDVI Time Series', vAxis: {title: 'NDVI'}, hAxis: {title: 'Date', format: 'MM-yyyy', gridlines: {count: 7}}, lineWidth: 1, pointSize: 3, series: { 0: {color: 'green'} } }); // Print the chart to the console print(chart); // Add the most recent image with NDVI to the map var recent = withNDVI.sort('system:time_start', false).first(); var visParams = { bands: ['B4', 'B3', 'B2'], min: 0, max: 0.3 }; Map.centerObject(point, 10); Map.addLayer(recent, visParams, 'Recent Landsat Image'); // Add NDVI layer var ndviVis = { min: 0, max: 1, palette: ['red', 'yellow', 'green'] }; Map.addLayer(recent.select('NDVI'), ndviVis, 'NDVI');
// Define a region of interest var roi = ee.Geometry.Rectangle([-122.5, 37.2, -121.5, 38.0]); // Get Landsat 8 collection var collection = ee.ImageCollection('LANDSAT/LC08/C02/T1_TOA') // Filter by date, cloud cover, and region .filterDate('2020-06-01', '2020-09-30') .filterBounds(roi) .filter(ee.Filter.lt('CLOUD_COVER', 10)); // Sort by cloud cover and get the least cloudy image var image = collection.sort('CLOUD_COVER').first(); // Calculate NDVI var ndvi = image.normalizedDifference(['B5', 'B4']).rename('NDVI'); // Create a mask for vegetation (NDVI > 0.4) var vegMask = ndvi.gt(0.4); // Apply the mask to the NDVI image var vegNdvi = ndvi.updateMask(vegMask); // Define visualization parameters var ndviVis = { min: 0, max: 1, palette: ['red', 'yellow', 'green'] }; var rgbVis = { bands: ['B4', 'B3', 'B2'], min: 0, max: 0.3 }; // Center the map on the region of interest Map.centerObject(roi, 9); // Add layers to the map Map.addLayer(image, rgbVis, 'Landsat 8 RGB'); Map.addLayer(ndvi, ndviVis, 'NDVI'); Map.addLayer(vegNdvi, ndviVis, 'Vegetation (NDVI > 0.4)'); // Add a legend var legend = ui.Panel({ style: { position: 'bottom-left', padding: '8px 15px' } }); var legendTitle = ui.Label({ value: 'NDVI', style: { fontWeight: 'bold', fontSize: '18px', margin: '0 0 4px 0', padding: '0' } }); legend.add(legendTitle); var makeRow = function(color, name) { var colorBox = ui.Label({ style: { backgroundColor: color, padding: '8px', margin: '0 0 4px 0' } }); var description = ui.Label({ value: name, style: {margin: '0 0 4px 6px'} }); return ui.Panel({ widgets: [colorBox, description], layout: ui.Panel.Layout.Flow('horizontal') }); }; legend.add(makeRow('red', '0')); legend.add(makeRow('yellow', '0.5')); legend.add(makeRow('green', '1.0')); Map.add(legend);
Landsat Guide

Landsat Collections in Earth Engine

  • Landsat 1-5 MSS: 1972-1999, 60m resolution, 4 bands
  • Landsat 4-5 TM: 1982-2012, 30m resolution, 7 bands
  • Landsat 7 ETM+: 1999-present, 30m resolution, 8 bands
  • Landsat 8 OLI/TIRS: 2013-present, 30m resolution, 11 bands
  • Landsat 9 OLI-2/TIRS-2: 2021-present, 30m resolution, 11 bands

Earth Engine provides access to different processing levels of Landsat data, including Top of Atmosphere (TOA) reflectance, Surface Reflectance (SR), and raw Digital Numbers (DN). The Collection 2 (C02) data is the latest reprocessed version with improved calibration and geometric accuracy.

Sentinel Analysis

The Sentinel missions are part of the European Union's Copernicus Program. Earth Engine provides access to Sentinel-1 (radar), Sentinel-2 (optical), and Sentinel-5P (atmospheric) data.

// Define a region of interest var roi = ee.Geometry.Point([13.4050, 52.5200]).buffer(10000); // Berlin // Get Sentinel-2 surface reflectance collection var collection = ee.ImageCollection('COPERNICUS/S2_SR') // Filter by date, cloud cover, and region .filterDate('2020-06-01', '2020-09-30') .filterBounds(roi) .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 20)); // Function to mask clouds using the SCL band function maskS2clouds(image) { // Bits 3 and 8-9 are cloud shadow and cloud var cloudBitMask = 1 << 3; var cirrusBitMask = 1 << 10; // Get the pixel QA band var qa = image.select('SCL'); // Both flags should be set to zero, indicating clear conditions var mask = qa.neq(3).and(qa.neq(9)).and(qa.neq(8)).and(qa.neq(10)); return image.updateMask(mask); } // Apply cloud mask var filtered = collection.map(maskS2clouds); // Create a median composite var composite = filtered.median(); // Define visualization parameters for true color var visParams = { bands: ['B4', 'B3', 'B2'], min: 0, max: 3000 }; // Define visualization parameters for false color (NIR) var visFalseColor = { bands: ['B8', 'B4', 'B3'], min: 0, max: 3000 }; // Center the map on the region of interest Map.centerObject(roi, 10); // Add layers to the map Map.addLayer(composite, visParams, 'Sentinel-2 True Color'); Map.addLayer(composite, visFalseColor, 'Sentinel-2 False Color (NIR)'); // Calculate NDVI var ndvi = composite.normalizedDifference(['B8', 'B4']).rename('NDVI'); var ndviVis = { min: 0, max: 0.8, palette: ['red', 'yellow', 'green'] }; Map.addLayer(ndvi, ndviVis, 'NDVI'); // Add a panel with layer controls var panel = ui.Panel({ style: { position: 'top-right', padding: '8px 15px' } }); var title = ui.Label({ value: 'Layer Controls', style: {fontWeight: 'bold', fontSize: '16px', margin: '0 0 10px 0'} }); panel.add(title); // Add layer visibility toggles function makeCheckbox(label, layer, defaultValue) { var checkbox = ui.Checkbox({ label: label, value: defaultValue, onChange: function(value) { Map.layers().get(layer).setShown(value); } }); return checkbox; } panel.add(makeCheckbox('True Color', 0, true)); panel.add(makeCheckbox('False Color (NIR)', 1, false)); panel.add(makeCheckbox('NDVI', 2, false)); Map.add(panel);
// Define a region of interest var roi = ee.Geometry.Point([13.4050, 52.5200]).buffer(20000); // Berlin // Get Sentinel-1 SAR GRD collection var collection = ee.ImageCollection('COPERNICUS/S1_GRD') // Filter by date, polarization, and region .filterDate('2020-01-01', '2020-12-31') .filterBounds(roi) .filter(ee.Filter.eq('instrumentMode', 'IW')) .filter(ee.Filter.listContains('transmitterReceiverPolarisation', 'VV')) .filter(ee.Filter.listContains('transmitterReceiverPolarisation', 'VH')); // Function to process SAR data function processSAR(image) { // Select VV and VH polarizations var vv = image.select('VV'); var vh = image.select('VH'); // Calculate VH/VV ratio (useful for vegetation and land cover) var ratio = vh.divide(vv).rename('VH_VV_ratio'); // Add ratio band to the image return image.addBands(ratio); } // Apply processing to each image var processed = collection.map(processSAR); // Create a median composite var composite = processed.median(); // Define visualization parameters var visVV = { bands: ['VV'], min: -25, max: 0 }; var visVH = { bands: ['VH'], min: -25, max: 0 }; var visRatio = { bands: ['VH_VV_ratio'], min: 0.3, max: 0.8, palette: ['blue', 'purple', 'cyan', 'green', 'yellow', 'red'] }; // Center the map on the region of interest Map.centerObject(roi, 9); // Add layers to the map Map.addLayer(composite, visVV, 'Sentinel-1 VV'); Map.addLayer(composite, visVH, 'Sentinel-1 VH'); Map.addLayer(composite, visRatio, 'VH/VV Ratio'); // Add a panel with information var panel = ui.Panel({ style: { position: 'bottom-left', padding: '8px 15px', width: '300px' } }); var title = ui.Label({ value: 'Sentinel-1 SAR Analysis', style: {fontWeight: 'bold', fontSize: '16px', margin: '0 0 10px 0'} }); panel.add(title); var info = ui.Label({ value: 'VV: Vertical transmit, vertical receive\n' + 'VH: Vertical transmit, horizontal receive\n' + 'VH/VV ratio: Useful for vegetation mapping', style: {whiteSpace: 'pre', fontSize: '12px'} }); panel.add(info); Map.add(panel);
Sentinel Guide

Sentinel Missions in Earth Engine

  • Sentinel-1: C-band Synthetic Aperture Radar (SAR) data, useful for all-weather, day and night observations
  • Sentinel-2: High-resolution optical imagery with 13 spectral bands at 10m, 20m, and 60m resolution
  • Sentinel-5P: Atmospheric monitoring data for air quality and climate

Sentinel-2 provides higher spatial resolution (10m) than Landsat (30m) for visible and NIR bands, making it excellent for detailed land cover mapping and monitoring. Sentinel-1 provides radar data that can penetrate clouds and operate in darkness, making it valuable for monitoring areas with persistent cloud cover.

Land Cover Classification

Earth Engine provides machine learning algorithms for supervised and unsupervised classification of satellite imagery, enabling land cover mapping and change detection.

// Define a region of interest var roi = ee.Geometry.Rectangle([-122.5, 37.5, -122.0, 38.0]); // Get Landsat 8 collection var collection = ee.ImageCollection('LANDSAT/LC08/C02/T1_TOA') // Filter by date, cloud cover, and region .filterDate('2020-06-01', '2020-09-30') .filterBounds(roi) .filter(ee.Filter.lt('CLOUD_COVER', 10)); // Create a composite var composite = collection.median(); // Select bands for classification var bands = ['B2', 'B3', 'B4', 'B5', 'B6', 'B7']; var image = composite.select(bands); // Define training data classes var urban = ee.FeatureCollection([ ee.Feature(ee.Geometry.Point([-122.3, 37.8]), {'class': 0}), ee.Feature(ee.Geometry.Point([-122.2, 37.7]), {'class': 0}) ]); var water = ee.FeatureCollection([ ee.Feature(ee.Geometry.Point([-122.4, 37.9]), {'class': 1}), ee.Feature(ee.Geometry.Point([-122.5, 37.8]), {'class': 1}) ]); var forest = ee.FeatureCollection([ ee.Feature(ee.Geometry.Point([-122.2, 37.9]), {'class': 2}), ee.Feature(ee.Geometry.Point([-122.1, 37.8]), {'class': 2}) ]); var agriculture = ee.FeatureCollection([ ee.Feature(ee.Geometry.Point([-122.1, 37.7]), {'class': 3}), ee.Feature(ee.Geometry.Point([-122.0, 37.6]), {'class': 3}) ]); // Merge all training data var trainingData = urban.merge(water).merge(forest).merge(agriculture); // Sample the input imagery to get training data var training = image.sampleRegions({ collection: trainingData, properties: ['class'], scale: 30 }); // Train a Random Forest classifier var classifier = ee.Classifier.smileRandomForest(50).train({ features: training, classProperty: 'class', inputProperties: bands }); // Classify the image var classified = image.classify(classifier); // Define visualization parameters var visParams = { min: 0, max: 3, palette: ['red', 'blue', 'green', 'yellow'] }; // Center the map on the region of interest Map.centerObject(roi, 10); // Add layers to the map Map.addLayer(composite, {bands: ['B4', 'B3', 'B2'], min: 0, max: 0.3}, 'Landsat 8 Composite'); Map.addLayer(classified, visParams, 'Land Cover Classification'); // Add a legend var legend = ui.Panel({ style: { position: 'bottom-right', padding: '8px 15px' } }); var legendTitle = ui.Label({ value: 'Land Cover Classes', style: { fontWeight: 'bold', fontSize: '16px', margin: '0 0 4px 0', padding: '0' } }); legend.add(legendTitle); var makeRow = function(color, name) { var colorBox = ui.Label({ style: { backgroundColor: color, padding: '8px', margin: '0 0 4px 0' } }); var description = ui.Label({ value: name, style: {margin: '0 0 4px 6px'} }); return ui.Panel({ widgets: [colorBox, description], layout: ui.Panel.Layout.Flow('horizontal') }); }; legend.add(makeRow('red', 'Urban')); legend.add(makeRow('blue', 'Water')); legend.add(makeRow('green', 'Forest')); legend.add(makeRow('yellow', 'Agriculture')); Map.add(legend); // Add accuracy assessment var validation = classified.sampleRegions({ collection: trainingData, properties: ['class'], scale: 30 }); var testAccuracy = validation.errorMatrix('class', 'classification'); print('Confusion Matrix:', testAccuracy); print('Overall Accuracy:', testAccuracy.accuracy()); print('Kappa Coefficient:', testAccuracy.kappa());
Classification Guide

Classification Methods in Earth Engine

  • Supervised Classification: Uses labeled training data to classify images
  • Unsupervised Classification: Identifies patterns in data without training data
  • Available Algorithms:
    • Random Forest
    • Support Vector Machine (SVM)
    • Classification and Regression Trees (CART)
    • Naive Bayes
    • K-means clustering

Classification in Earth Engine can be used for land cover mapping, change detection, crop type identification, urban growth monitoring, and many other applications. The platform's ability to process large datasets makes it possible to create classifications at regional or global scales.
